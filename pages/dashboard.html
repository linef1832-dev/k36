help_outline คำอธิบายสิทธิ์การใช้งาน

  • สิทธิ์เมนูหลัก: หากติ๊กถูก พนักงานจะเห็นปุ่มเมนูนั้นแถบด้านซ้ายมือ (เช่น หน้าสรุปยอด, คลังรูปภาพ)
  • สิทธิ์ที่มี >> นำหน้า: คือสิทธิ์ "แอดมิน" สำหรับเมนูนั้นๆ เช่น
    - >> แอดมินจัดเวร: ทำให้พนักงานคนนั้นกดสุ่มจัดเวรหรือแก้ไขตารางได้
    - >> แอดมินคลังไฟล์: ทำให้พนักงานคนนั้นเพิ่ม/ลบไฟล์ ในหน้าคลังไฟล์ได้
    - >> ย้ายห้อง / เช็คชื่อ: เป็นสิทธิ์เฉพาะสำหรับเครื่องมือ Discord
  • หมายเหตุ: ผู้จัดการ (Manager / Admin) จะมีสิทธิ์เห็นทุกเมนูและกดได้ทุกปุ่มอัตโนมัติ โดยไม่ต้องมาติ๊กในหน้านี้
