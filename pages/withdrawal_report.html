help_outline วิธีตั้งค่าบอท

  1. เปิด Telegram → ค้นหา @BotFather
  2. พิมพ์ /newbot → ตั้งชื่อบอท → รับ Token
  3. เพิ่มบอทเข้ากลุ่มที่ต้องการ และให้สิทธิ์ Admin
  4. นำ Token มาใส่ในช่องด้านบน แล้วกดทดสอบ
  5. รัน tg_bot.py บน Server/VPS ของคุณ
